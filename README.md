# Windows 11 Setup Script

Setup scripts for Windows 11 PC.

How to use:

    1 - Install Windows 
    2 - Setup User Account
    3 - Rename PC to wanted name
    2 - Login
    3 - Open Edge and download this repo
    4 - Run PS as Admin
    5 - Execute 
        Set-ExecutionPolicy -ExecutionPolicy Bypass -Scope Process -Force
    6 - Run the script
        ./setup.ps1
